Domain Controller Setup

Go to Server Manager and add two roles: Active Directory Domain Services and DNS Server

My Configuration:

  • Once that's done, click "Promote this server to a domain controller"
  • "Add a new forest"
  • Named DC : test.lab
  • Set a restore mode password

Promote to Domain Controller:

In Server Manager, click notification flag (yellow warning triangle)

  • Click "Promote this server to a domain controller"
  1. Deployment:
    • Select "Add a new forest"
    • Enter Root domain name: "corp.contoso.com"
  1. Domain Controller Options:
    • "Global Catalog (GC)"
    • Enter Directory Services Restore Mode password
  1. DNS Options:
    • Ignore DNS delegation warning
  1. Additional Options:
    • NetBIOS domain name: CORP (auto-filled)
  1. Paths:
    • Keep default paths for Database, Log files, and SYSVOL
  1. Review Options:
    • Review selections
  1. Prerequisites Check:
    • Wait for prerequisite check to complete
  1. restart

Login

Login to the new domain

  1. Note the login shows: "Sign in to: X"
  1. Username is now: X\Administrator
    1. Enter the Administrator password
  • "Domain Name System (DNS) server"
